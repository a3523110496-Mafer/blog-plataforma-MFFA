# Plataforma Web de Publicaciones

Aplicación web orientada a la creación y gestión de publicaciones, desarrollada como un proyecto con enfoque profesional, arquitectura de información clara y principios de accesibilidad.

## Problema que resuelve
Muchas personas y comunidades dependen de redes sociales para publicar contenido, lo que limita el control y la organización de la información. Esta plataforma permite centralizar publicaciones en un sistema propio y estructurado.

## Usuarios
- Visitantes: pueden visualizar publicaciones.
- Usuarios registrados: pueden crear y editar publicaciones.
- Administrador: gestiona contenido y usuarios.

## Alcance inicial (MVP)
- Registro e inicio de sesión
- Visualización pública de publicaciones
- Creación y edición de publicaciones
- Panel privado básico
- Navegación accesible por teclado

## Tecnologías
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Infraestructura: Docker, GitHub Actions
- Control de versiones: Git y GitHub

## Accesibilidad
El sistema está diseñado para permitir navegación por teclado y uso sin mouse, respetando el orden de tabulación y jerarquías de contenido.

## Estructura del proyecto
- frontend/: interfaz de usuario
- backend/: lógica del sistema
- docs/: documentación del proyecto
